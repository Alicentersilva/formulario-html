# Cadastro de Alunos

Projeto desenvolvido durante a graduação em Engenharia de Software com o objetivo de praticar a criação de formulários web utilizando HTML e CSS.

## Tecnologias utilizadas

* HTML5
* CSS3

## Funcionalidades

* Cadastro de nome e sobrenome
* Campo de e-mail
* Seleção de turno
* Escolha de série escolar
* Seleção de atividades extracurriculares
* Campo de mensagem
* Validação de campos obrigatórios

## Objetivo

Aplicar conceitos fundamentais de desenvolvimento web, incluindo formulários, elementos de entrada de dados, estilização com CSS e organização de interfaces para o usuário.
